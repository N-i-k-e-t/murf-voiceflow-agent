Welcome to VoiceFlow

Talk to your terminal. Press Enter, speak, and hear Murf Falcon respond. Built by Niket Patil.

Features

  • Deepgram ASR for accurate speech-to-text
  • OpenAI LLM for concise, helpful replies
  • Murf Falcon streaming TTS for natural voice
  • Responsive landing site with enquiry form
  • Deployable to Vercel with serverless API
